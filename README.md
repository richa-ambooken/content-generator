# The Problem

Marketing teams often struggle with repeatedly rewriting the same content for blogs, social media, and emails. This process is time-consuming, leads to inconsistent messaging, and increases the risk of factual errors and creative fatigue.

# The Solution

This project introduces a multi-agent AI system that automates content creation by simulating a real content team. A research agent extracts accurate information, a copywriter agent generates multi-format content, and an editor agent ensures quality and correctness. Additional features like a chatbot for corrections and a live dashboard make the system interactive, efficient, and reliable.
